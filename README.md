# AI CRM HCP Assistant

## Tech Stack
- React.js
- FastAPI
- LangGraph
- Groq LLM
- SQLite

## Features
- Log HCP Interaction
- Edit Interaction
- Delete Interaction
- AI Chat Assistant
- AI Summarization
- AI Recommendations
- AI Follow-up Planner
- Search Interaction
- Analytics Dashboard

## Run Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```
